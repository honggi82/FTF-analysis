## 알고리즘: F값 시간-주파수(FTF) 분석

FTF 분석은 분산 분석(ANOVA)의 F 값을 적용하여 시간-주파수 맵에서 통계적 유의성을 시각화하는 새로운 기술입니다. 뇌파검사(EEG)와 같은 시계열 데이터에서 여러 조건 간의 차이를 식별하고 정량화하도록 설계되었습니다.

### 주요 기능 및 방법론
* **통계적 시각화**: 전력이나 진폭만 표시하는 기존의 시간-주파수 플롯과 달리 FTF 플롯은 각 시간-주파수 지점에서 F-값을 직접 시각화합니다. 이를 통해 연구자들은 실험 조건 간의 가장 중요한 차이점이 어디에 있는지 즉시 확인할 수 있습니다.
* **분산 분석**: 이 방법의 핵심은 조건 간 분산과 조건 내 분산의 비율을 계산하는 것입니다. F 값이 높다는 것은 조건 간의 변동이 각 조건 내의 변동보다 상당히 크다는 것을 의미하며 이는 실제 효과를 나타냅니다.
* **직관적인 해석**: 결과 FTF 맵은 조건에 따라 신호가 크게 달라지는 특정 시점과 주파수 대역을 강조합니다. 이는 파워 맵 자체에 대한 사후 통계 테스트 없이도 운동 이미지 작업과 같은 복잡한 신경 신호를 분석하기 위한 직관적이고 강력한 도구를 제공합니다.

이 저장소의 MATLAB 코드에는 시계열 데이터에 대한 FTF 분석을 수행하는 데 필요한 함수와 예제 스크립트가 포함되어 있습니다.
