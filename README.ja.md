このリポジトリは、論文「F 値時間周波数 (FTF) 解析: 相互内分散解析」で提案されている F 値時間周波数 (FTF) 解析手法の MATLAB コードを提供します。

## アルゴリズム: F 値時間周波数 (FTF) 解析

FTF 分析は、分散分析 (ANOVA) からの F 値を適用することにより、時間-周波数マップ上の統計的有意性を視覚化する新しい手法です。脳波検査 (EEG) などの時系列データ内の複数の条件間の違いを特定し、定量化するように設計されています。

### 主な機能と方法論
* **統計的視覚化**: パワーまたは振幅のみを表示する従来の時間-周波数プロットとは異なり、FTF プロットは各時間-周波数ポイントでの F 値を直接視覚化します。これにより、研究者は実験条件間の最も大きな違いがどこにあるのかをすぐに特定することができます。
* **分散分析**: このメソッドの核心は、条件間の分散と条件内の分散の比率を計算することです。高い F 値は、条件間の変動が各条件内の変動よりも大幅に大きいことを示し、真の効果を示唆しています。
* **直感的な解釈**: 結果として得られる FTF マップは、信号が条件全体で大きく発散する特定の時点と周波数帯域を強調表示します。これにより、パワー マップ自体に対する事後統計テストを必要とせずに、運動イメージタスクからの信号などの複雑な神経信号を分析するための直感的で強力なツールが提供されます。

このリポジトリの MATLAB コードには、独自の時系列データに対して FTF 解析を実行するために必要な関数とサンプル スクリプトが含まれています。
