Этот репозиторий предоставляет код MATLAB для метода частотно-временного анализа F-значения (FTF), предложенного в статье «Анализ частотно-временных значений F (FTF): анализ внутренних отклонений».

## Алгоритм: частотно-временной анализ F-значения (FTF)

Анализ FTF — это новый метод, который визуализирует статистическую значимость на частотно-временной карте путем применения значения F из дисперсионного анализа (ANOVA). Он предназначен для выявления и количественной оценки различий между несколькими состояниями в данных временных рядов, таких как электроэнцефалография (ЭЭГ).

### Ключевые особенности и методология
* **Статистическая визуализация**. В отличие от традиционных графиков «время-частота», которые показывают только мощность или амплитуду, график FTF напрямую визуализирует значение F в каждой частотно-временной точке. Это позволяет исследователям сразу определить, в чем заключаются наиболее существенные различия между условиями эксперимента.
* **Анализ отклонений**. Суть метода заключается в вычислении отношения дисперсии между условиями к дисперсии внутри условий. Высокое значение F указывает на то, что вариация между условиями значительно больше, чем вариация внутри каждого условия, что указывает на истинный эффект.
* **Интуитивная интерпретация**: полученная карта FTF выделяет конкретные моменты времени и диапазоны частот, где сигналы значительно расходятся в зависимости от условий. Это обеспечивает интуитивно понятный и мощный инструмент для анализа сложных нейронных сигналов, например сигналов, возникающих в задачах по воображению движений, без необходимости проведения апостериорных статистических тестов на самих картах мощности.

Код MATLAB в этом репозитории включает функции и примеры сценариев, необходимые для выполнения анализа FTF ваших собственных данных временных рядов.
